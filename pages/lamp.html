LE SERVEUR LAMP

Pourquoi installer un serveur HTTP ?

Poourquoi installer votre propre serveur HTTP ? le side rendering est disponible partout. Cest un outil util qui m'a permis de me familiariser avec Ubuntu.

Le serveur Apache s'installe très simplement sur Ubuntu. Le fait de pouvoir mettre en ligne son site, sans avoir à passer par un hébergeur aide à comprendre le fonctionnement des noms de domaine et des serveurs, comment obtenir un nom de domaine et l'affilier à un dossier de production.

Le serveur LAMP regroupe 4 technologies, acronyme de Linux, Apache, MYsql et PHP, son installation nécessite donc d'avoir déjà le système d'exploitation Ubuntu/Linux présent. On installe ensuite Apache, il y a beaucoup de tuto disponible, je vais vous expliquer comment l'installer, le configurer et le mettre à jour.

Par la suite la création d'un hôte virtuel vous permettra d'héberger votre site sous votre nom de domaine. Pour finir on installe un SGBD (service de gestion de base de données) et PHP, et votre serveur peut maintenant héberger des sites dynamiques.

Qu'est ce qu'un serveur HTTP ?

Un serveur HTTP est un ordinateur qui héberge et diffuse du contenu, site web, sur les navigateurs à leurs demandes (requêtes http)
Lorsque vous tapez une adresse, un nom de domaine, dans le navigateur, lui va chercher une adresse ip. C'est une série de chiffres semblable à un code barre.
Les noms de domaine sont fournis par des serveurs DNS qui associent les adresses ip à des noms de domaines propriétaires.
Le navigateur envoie une requête à l'adresse que vous lui avez demandé, cette requête sera reçu par un serveur HTTP, en retour vous recevrez sur votre navigateur le contenu demandé.

HTML et CSS

Un navigateur web n'est capable de lire uniquement HTML, CSS, et JavaScript ce sont 3 langages qu'il est capable de comprendre d'interpréter et de traduire, ce sont les languages Front End, Apache seul ne pourra diffuser que des fichiers html et css. Javascript nécessitera un serveur différent. HTML et CSS sont des langages très puissant et irremplaçable au sein du développement web, il correspondent à tout le contenu et tout le style de texte, les couleurs, les polices de caractères, taille, fond d'écran, etc... Que vous pouvez voir sur les sites.

PHP

Le rôle de PHP sera le lien avec un serveur de base de données, il permet de générer du contenu html qui sera fournit au serveur HTTP. Il n'est pas visible coté navigateur, il agira uniquement coté serveur, c'est le language Back End. Ainsi on crée des sites dynamiques interactifs, évolutions des sites statiques uniquement lisibles.
