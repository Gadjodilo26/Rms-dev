Bonjour visiteur

Je suis Arnaud, actuellement inscris dans un cursus d'apprentissage du métier de développeur, c'est une formation à distance proposée par STUDI, très bonne école que je recommande.
Ce site évoluera donc toute au long de ma formation. J'en suis actuellement à 4 mois j'ai débuté en novembre 2022.

Je commence à maîtriser HTML, CSS, ce qui m'a permis de démarrer la construction de ce site que j'héberge à mon domicile sur un serveur apache.
Le site évoluera en PHP par la suite, avec les possibilités que cela offrira.

Pour le contenu, je proposerai au fur et à mesure des éléments annexes au développement, je ne me ferai pas passer pour un formateur en développement web, j'expliquerai simplement comment j'ai mis en place mon environnement de développement, ce qui me parait tès util quand on veut se lancer.

Un exemple simple, qui est le premier sujet de ce site, est l'instalation d'un serveur LAMP et la découverte de Ubuntu et Linux, qui pour moi sont maintenant utilisés quotidiennement.

Le serveur HTTP Apache

Le logiciel libre Apache HTTP Server (Apache) est un serveur HTTP créé et maintenu au sein de la fondation Apache. Jusqu'en avril 2019, ce fut le serveur HTTP le plus populaire du World Wide Web. Il est distribué selon les termes de la licence Apache.

Le SGBD MySQL

MySQL est un système de gestion de bases de données relationnelles (SGBDR). Il est distribué sous une double licence GPL et propriétaire. Il fait partie des logiciels de gestion de base de données les plus utilisés au monde, autant par le grand public (applications web principalement) que par des professionnels, en concurrence avec Oracle, PostgreSQL et Microsoft SQL Server.

PHP un lanquage coté serveur

PHP est un langage de script utilisé le plus souvent côté serveur:
dans cette architecture, le serveur interprète le code PHP des pages web demandées et génère du code (HTML, XHTML, CSS par exemple) et des données (JPEG, GIF, PNG par exemple) pouvant être interprétés et rendus par un navigateur web. PHP peut également générer d'autres formats comme le WML, le SVG et le PDF.
